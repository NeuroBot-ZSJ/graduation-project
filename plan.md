## Plan: 开题报告写作执行计划（按定稿大纲）

以你当前确认的大纲为唯一目录基准，沿用模板文档格式，仅替换为你的课题内容。整体研究思路采用三条相对独立但可联动验证的技术线：动力学参数辨识、冗余机械臂阻抗控制、碰撞检测与误报抑制。

**Steps**
1. Phase A 目录与章节边界锁定（阻塞后续）
2. 锁定第1章至第6章标题层级，不新增或删除主章节。
3. 明确“研究内容”只回答做什么，“主要研究方案”只回答怎么做，避免重复。
4. 将你补充的关键约束写入章节骨架：物理一致性约束、滤波预处理、动力学前馈、零空间投影误报抑制。
5. Phase B 第2章研究内容（做什么）
6. 2.2.1 物理一致约束下的动力学参数辨识研究：对 LS、WLS、GA、LS-GA、PSO 做统一对比，目标是获得可用于前馈补偿的稳定参数。
7. 2.2.2 冗余机械臂分层阻抗控制研究：任务空间笛卡尔阻抗控制 + 零空间关节空间阻抗控制，目标是兼顾作业性能与柔顺交互。
8. 2.2.3 碰撞检测与误报抑制研究：对比力矩残差、广义动量观测器、二阶广义动量观测器，研究动力学一致伪逆下的零空间投影抑制误报。
9. 2.3 关键问题与难点：参数物理可行性、多任务优先级求解稳定性、碰撞检测灵敏度与误报率平衡。
10. Phase C 第3章主要研究方案（怎么做）
11. 3.1.1 动力学辨识方案：
12. 数据预处理链：均值滤波、巴特沃斯低通滤波、零相位滤波处理q、dq、torque数据。
13. 方法对比链：LS、WLS、GA、WLS-GA、PSO 在统一数据与激励下评估。
14. 约束求解链：引入物理一致性约束（惯性矩阵对称正定等）并输出前馈补偿参数。
15. 3.1.2 阻抗控制方案：
16. 控制结构：笛卡尔空间阻抗为主任务，零空间关节阻抗为次任务。
17. 求解器方案：多任务优先级控制框架下对比 WQP、CQP、HQP。
18. 融合策略：叠加辨识得到的动力学前馈项，验证跟踪精度与交互性能提升。
19. 3.1.3 碰撞检测方案：
20. 检测器对比：力矩残差、广义动量观测器、二阶广义动量观测器。
21. 误报抑制：基于动力学一致伪逆进行零空间投影，抑制作业时末端任务引发的误报。
22. 评价指标：检测时延、误报率、漏报率、阈值鲁棒性。
23. Phase D 技术路线、可行性与创新点
24. 3.2 技术路线：数据采集与预处理 -> 参数辨识与物理一致约束 -> 分层阻抗控制与前馈融合 -> 碰撞检测与零空间投影 -> 统一仿真验证。
25. 3.3 可行性分析：理论可行、算法可行、平台可行（Panda + MuJoCo）、实验可复现可行。
26. 4.2 创新点采用“适度创新”表述：
27. 4.2.1 物理一致约束与滤波预处理融合的动力学辨识改进。
28. 4.2.2 冗余机械臂分层阻抗控制与动力学前馈融合实现。
29. 4.2.3 基于动力学一致零空间投影的碰撞检测误报抑制。
30. Phase E 收尾与检查
31. 目录一致性检查：目录标题、正文标题、研究内容与方案一一对应。
32. 术语一致性检查：统一使用“动力学参数辨识、笛卡尔阻抗、零空间阻抗、动力学一致伪逆、碰撞误报抑制”等术语。
33. 规范检查：对照学校要求文档检查格式、参考文献、章节完整性。
